# ✨ Y2K Portfolio - Digital Universe ✨

🚧 **PROJETO EM ANDAMENTO (WIP)** 🚧

Um portfólio pessoal com estética Y2K (anos 2000), misturando a vibe de internet antiga, colagens digitais, cultura de videogames retrô e o início do mundo digital moderno.

---

## 🎨 Características

* **Estética Y2K autêntica**: glitter digital, chrome/metalizado, efeitos holográficos
* **Paleta nostálgica**: rosa neon, azul elétrico, roxo digital e prata cromado
* **Elementos retrô**: janelas estilo Windows XP, ícones de pastas, vibe MSN/ICQ
* **Animações interativas**: loading de internet discada, efeitos glitch, cursor personalizado
* **Easter eggs escondidos**: código Konami e interações secretas
* **Totalmente responsivo**: funciona em desktop, tablet e mobile

---

y2k-portfolio/

## 🚀 Como Usar

### 📂 Abrir direto no navegador

1. Baixe ou clone o projeto
2. Abra o arquivo `index.html` no navegador
3. Pronto — não precisa de instalação

---

### ⚙️ Usar servidor local (recomendado)

#### Python

```bash
python -m http.server 8000
```

#### Node.js

```bash
npx http-server
```

#### VS Code (Live Server)

* Instale a extensão **Live Server**
* Clique com botão direito em `index.html`
* Selecione **Open with Live Server**
